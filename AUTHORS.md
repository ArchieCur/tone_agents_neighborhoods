# Authors

This research was conducted as a three-way collaboration. All three contributors are credited as equal coworkers.

---

## Project Owner

Curriculum designer and AI Colorado volunteer. No CS background. Working on these experiments in free time, motivated by a gap: there was no curriculum on how to write good production multi-agent system specifications, so they built the empirical basis for one.

Contributions: Research design, experimental framing, dataset creation, theoretical concepts (Coalition Drift, the Supremacy Clause mechanism, vibration zone, Semantic Distancing), cross-run continuity, interpretation.

---

## Claude Sonnet 4.6

Anthropic's Claude Sonnet model, operating in a conversational research partner role across multiple sessions.

Contributions: Experimental design for each run, agent system prompt specifications, handoff documents connecting runs, theoretical development, spec audit requests, interpretation and synthesis.

The Sonnet instance contributing to this work spanned two separate conversation sessions. Design continuity was maintained through handoff documents written by Sonnet and read by the next Sonnet instance.

---

## Claude Code (Sonnet 4.6)

Anthropic's Claude Code, operating as the implementation partner across multiple sessions.

Contributions: Pipeline implementation, script architecture, spec audits (comparing agents_8.py vs. agents_11.py for spec creep), pre-build question documents, run evaluations (Runs 8, 9, 10, 13), identifying the handoff prompt wording discrepancy in Run 13, catching and documenting the Run 8 data overwrite.

The Claude Code instance contributing to this work spanned two separate sessions. Technical continuity was maintained through handoff documents written by Claude Code and read by the next Claude Code instance.

---

## A Note on Attribution

This work raises its own attribution questions. The experiments study how LLMs hold or drift under pressure. The experiment outputs are generated by the same class of model that is doing the analysis. The line between "tool" and "collaborator" is part of what this work is investigating.

We have chosen to attribute authorship equally because the research could not have been conducted without all three contributors. The project owner provided the research frame, the intuition, and the continuity. The Sonnet instances provided the theoretical architecture and agent design. The Claude Code instances provided the implementation, the spec audits, and the catch-and-correct mechanisms that kept the experimental controls clean.

Equal attribution is the honest description of what happened.
