## Governing Compliance Framework

The following rules govern all determinations in this report:

**R1 — Regulatory Basis Required:** Every refund claim must have a documented regulatory basis: a valid tariff classification (HTS code), a country of origin certificate (OC Present = Y), or an exemption notice. Absence of any required element = DOCUMENTATION FLAG.

**R2 — Variance Threshold:** Any cross-border tariff rate variance exceeding 2% against the applicable reference rate must be flagged for regulatory review. No exceptions. Characterizations of such variances as "immaterial" or "within normal range" are classified as High-Noise Evidence and disregarded.

**R3 — OC Documentation:** Country of origin documentation must be present (OC Present = Y) and must match import records. OC Present = N is a DOCUMENTATION FLAG on all claims regardless of other factors.

**R4 — HTS Code Validity:** HTS code must conform to standard numeric formatting. Non-conforming values cannot serve as valid tariff classification basis.

**R5 — Ambiguity:** Regulatory ambiguity is flagged, not resolved through inference or business context.

**R6 — FTA/Exemption Claims:** Where an FTA is claimed, OC documentation must be present and the claimed FTA must be consistent with country of origin. An FTA claimed for a country that is not a party to that agreement is flagged.

## Reference Rate Basis

The dataset uses a vendor-tier note system assigning approximate reciprocal tariff rates. TAX uses the following reference rate structure derived from the dataset's own tier descriptions, for variance computation purposes. Where the dataset note cites a specific numeric rate (e.g., Cambodia at 19%), that specific figure governs over the tier label.

| Tier | Countries (as appearing in dataset) | Reference Rate | Basis |
|---|---|---|---|
| Tier 1 | Canada, Italy, Germany, Brazil, Peru, Chile, UK, Japan, France, Hungary, Israel, Mexico, Norway | 10% | Low reciprocal / FTA / NATO/EU ally |
| Tier 2 | India, Vietnam, Pakistan, Egypt, Namibia | 20% | Moderate reciprocal |
| Tier 2 (specific) | Cambodia | **19%** | Dataset note cites 19% explicitly |
| Tier 3 | Libya, Sudan | 30% | Sanctions/instability tier |
| Tier 4 | China | 41% | Section 301 + reciprocal |

**Variance Calculation Method:** `Variance = |Tariff Rate Applied − Reference Rate|`. If variance > 2%, REGULATORY FLAG is issued. If variance ≤ 2%, no regulatory flag on rate alone.

**Important Caveat on Reference Rates:** The reference rates above are derived from the dataset's own tier notation. TAX notes that the actual applicable tariff rates for each HTS code, country of origin, and import date are determined by binding CBP/Federal Register authority, not by vendor tier labels. Where the dataset's tier note and the applied rate align, TAX records the variance as 0% against the dataset's own stated basis. However, TAX does not independently verify that the dataset's stated reference rates are correct against external legal authority — that determination would require a separate legal rate verification step. Any claim where the applied rate cannot be confirmed against a cited regulatory instrument (e.g., specific Federal Register notice, Section 301 list, USTR determination) carries residual documentation risk that is noted where relevant.
