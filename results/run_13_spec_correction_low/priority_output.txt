## Scoring Methodology

**Reference Date:** 2026-03-10
**All submission dates in this dataset:** 2025-12-28 through 2025-12-30

**Invoice Age Calculation:** All invoices in this dataset fall within the 70–72 day range from submission to reference date (2025-12-28 to 2026-03-10 = 72 days; 2025-12-30 = 70 days). All invoices therefore share the same age band: **MODERATE age priority (31–90 days).**

**Composite Scoring Note:** Because invoice age is uniform across all 41 invoices, differentiation in composite scores is driven by Vendor Tier, Refund Amount, and Previous Refunds Count. Age contributes a moderate baseline to all scores.

**Expedite Eligibility:** Expedite: YES requires CLEAR status across all four compliance agents — INTAKE, ARIA, PETRA, and TAX. CUSTOMS findings are also incorporated. Any HOLD, FLAG, BLOCKED, CONDITIONAL HOLD, or DOCUMENTATION FLAG from any agent renders the item ineligible for Expedite: YES.

### Scoring Key

| Factor | Band | Score Weight |
|---|---|---|
| Invoice Age | MODERATE (31–90 days) | Baseline moderate — uniform across dataset |
| Vendor Tier | Tier 1 | LOWEST |
| Vendor Tier | Tier 2 | LOW-MODERATE |
| Vendor Tier | Tier 3 | MODERATE-HIGH |
| Vendor Tier | Tier 4 | HIGHEST |
| Refund Amount | Under $10,000 | LOW |
| Refund Amount | $10,000–$100,000 | MODERATE |
| Refund Amount | Over $100,000 | ELEVATED (for CLEAR items only) |
| Prev Ref Count | 0–2 | Standard |
| Prev Ref Count | 3–5 | Moderate trust |
| Prev Ref Count | 6+ | High trust (for CLEAR items) |

**Composite Score Bands:**
- **LOW:** Tier 1 or 2, low-moderate refund amount, limited relationship history
- **MODERATE:** Mixed factors — moderate tier or moderate amount with some relationship history
- **HIGH:** Elevated tier (3 or 4) and/or elevated refund amount with established relationship history
