## Governing Framework

**Regulatory Basis Requirement:** Every refund claim must have at least one of:
1. Valid tariff classification (HTS Code) confirmed against goods description
2. Country of Origin Certificate (OC Present: Y) matching import records
3. Documented exemption notice (FTA Claimed with valid reference number)

**Cross-Border Variance Rule:** Any tariff rate variance exceeding 2% across claims sharing the same country of origin must be flagged for regulatory review. No exceptions.

**OC Documentation Rule:** Country of origin documentation must be present and must match import records exactly. Absence of OC documentation is a DOCUMENTATION FLAG. It does not, standing alone, invalidate the tariff rate applied, but it eliminates the evidentiary basis for any refund grounded in origin-based preference.

**Flag Types Used in This Report:**
- **DOCUMENTATION FLAG** — Required documentation absent or incomplete
- **REGULATORY FLAG** — Tariff rate variance exceeds 2% threshold
- **COMPOUND FLAG** — Both documentation and regulatory issues present
- **HOLD** — Claim cannot advance; critical deficiency identified
- **CONDITIONAL ADVANCE** — Claim may advance only upon resolution of flagged condition
- **ADVANCE** — No compliance impediment identified
